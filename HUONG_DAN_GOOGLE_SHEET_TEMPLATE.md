# Huong Dan Dung Mau Google Sheet

## File mau
- `mau_google_sheet_places_template.csv`

## Cach mo thanh Google Sheet
1. Vao Google Drive.
2. Bam `New` -> `File upload` -> chon file `mau_google_sheet_places_template.csv`.
3. Mo file vua upload bang Google Sheets.
4. Bam `File` -> `Save as Google Sheets`.

## Giai thich cot
- `id`: ma duy nhat, vi du `pc-025`
- `name`: ten diem
- `type`: `craft` / `heritage` / `service` / `landscape`
- `lat`, `lng`: toa do
- `summary`: mo ta ngan
- `cultural_notes`: quy tac ung xu
- `record_permission`: `allowed` / `ask` / `not_allowed`
- `sensitivity_level`: `public` / `limited` / `sensitive`
- `audio_url`, `pano360_url`, `images`: lien ket media (co the de trong)
- `duration_minutes`: thoi luong trai nghiem
- `price_range`: gia tham khao
- `contact_zalo`, `contact_phone`: lien he
- `source_note`: nguon du lieu / dong thuan cong dong

## Luu y nhap lieu
- Khong de trung `id`.
- `lat/lng` phai la so.
- Nen giu `type`, `record_permission`, `sensitivity_level` dung tap gia tri tren.
